## 2. 배경

- EXPERT 과제는 Jetson 기반 자율 방재 에이전트로 비화재보를 줄이는 **현장 실증** 과제였다(2026-04-09 제안서 제출, 이후 탈락).
- 제안서의 핵심 KPI(자율 판정 정확도 80%, 오경보 억제율 50%, 운영 DB 200건 등)는 **현장 실증을 전제한 목표값**이라 측정된 근거가 없어 논문화가 불가능하다.
- 그러나 제안 준비 과정에서 화재 탐지 모델에 대한 **완결된 ablation 실험**을 실제로 수행하였고, 이 결과는 그 자체로 논문 1편이 된다.

## 3. 논문의 정체성

- **제목(가제)**: 비화재보 저감을 위한 엣지 화재·연기 탐지 모델의 데이터 구성 효과 분석
- **핵심 주장**: 화재 탐지 성능은 **모델 크기가 아니라 데이터 구성**(클래스 균형 + 정상배경 포함)이 결정한다.
  - 클래스 균형(FL:SM 1:1): +7.5%p
  - 정상배경(NM) 추가: +7.5%p
  - 모델 용량 확대(11n→11s): +0.7%p
  - → 데이터 요인이 모델 요인보다 약 **10배** 크다.
- **최종 성능**: YOLO11n, mAP@0.5 = 0.911, Jetson Orin Nano TensorRT FP16 ~47 FPS
- **데이터**: AIHub 071751 (공개 데이터, NDA·사생활 무관 → 외부 공개 안전)

## 4. 실험 결과 (실측, 이미 완료됨)

수치 원본(SSOT) = `TRAINING_LOG.md`. 본 표는 그 요약이다.

### 실험 A — 데이터 구성 (모델 고정: YOLO11n)

| 구성 | mAP@0.5 | 핵심 |
|------|---------|------|
| 화염 단독 | 0.777 | 연기 탐지 불가(재현율 0.640) |
| 불균형 14:1 | 0.761 | 소량 연기는 오히려 악화 |
| 균형 1:1 | 0.836 | smoke AP 0.684→0.792 급등 |
| 균형+정상 | **0.911** | 정밀도 0.888·재현율 0.853 동시 달성 |

### 실험 B — 모델 용량 (데이터 고정: 균형+정상)

| 모델 | mAP@0.5 |
|------|---------|
| YOLO11n | 0.911 |
| YOLO11s | 0.918 |
| YOLO11m | 0.914 |
| YOLO11l | 발산 실패 |

### 부가 결과
- 입력 해상도 640→416: 정확도 손실 ~0.4%p(거의 무손실), 엣지 처리량 약 2배
