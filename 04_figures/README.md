# 04_figures — 그림 출처 안내

> ★ 중요: 아래 그림은 모두 **참고·예시용**이다. 논문 본문의 **결과 그림(혼동행렬·PR곡선·정성 탐지 결과)은 여기에 없다.** 결과 그림은 학습 가중치(best.pt)가 삭제되어 현재 재현 불가하며, 재학습(인계문서 6절 보강 옵션 B)으로 생성해야 한다.

## 파일별 출처와 용도

| 파일 | 출처 | 논문에서의 위치 |
|------|------|----------------|
| `dfire_examples.png` | DFire 공개 데이터셋 샘플 이미지 | 데이터셋 소개 절 예시로 사용 가능 |
| `detection_01.png`~`detection_03.png` | 데모 시스템 탐지 출력(데모용 가중치 `yolo11n_smoke.pt` 기반) | **참고용.** 논문 결과 가중치 출력 아님 → 정성 예시로 쓰려면 재학습 모델로 다시 생성 권장 |
| `detection_grid.png` | 위 탐지 출력 그리드 | 동일(참고용) |
| `arch_diagram.png` | **탈락한 EXPERT 제안서의 자율 에이전트 시스템 구조도** | ⚠ 본 논문(데이터 구성 ablation)과 **무관**. 논문에 넣지 말 것. 시스템 맥락 참고용으로만 보관 |

## 논문에 필요하지만 현재 없는 그림 (재학습 시 생성)

| 그림 | 생성 방법 |
|------|----------|
| 혼동행렬(confusion matrix) | 재학습 best.pt → `03_scripts/` val() 산출물 |
| PR 곡선 / F1 곡선 | 동일 |
| 데이터 구성별 mAP 막대그래프 | `02_data_ssot/TRAINING_LOG.md` 수치로 직접 작도 가능(가중치 불요) |
| 모델 용량별 mAP 비교 | 동일(수치로 작도 가능) |
| 정성 탐지 예시(재학습 모델) | 재학습 best.pt 추론 |

> 데이터 구성·모델 용량 비교 막대그래프는 가중치 없이 `TRAINING_LOG.md` 표 수치만으로 바로 그릴 수 있다. 혼동행렬·PR곡선·정성예시는 가중치 재학습이 전제다.
