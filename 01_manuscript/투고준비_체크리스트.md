## 1. 논문 정체성

- **주제**: 비화재보 저감을 위한 엣지 화재·연기 탐지 모델의 데이터 구성 효과 분석
- **핵심 주장**: 데이터 구성(클래스 균형 +7.5%p, 정상배경 추가 +7.5%p)이 모델 용량(+0.7%p)보다 약 10배 결정적이다.
- **데이터**: AIHub 071751 (공개, NDA·사생활 무관) — private-data-disclosure 규칙 위반 0
- **현장 데이터 미포함**: 강남요양원 등 실증 데이터가 안 들어가서 오히려 깨끗함

## 2. 지금 투고 가능한 부분 (재실행 불필요)

| 요소 | 상태 |
|------|------|
| 초록(국·영문) | ✅ 완성 |
| 서론·관련연구·방법 | ✅ 완성 |
| 결과 표 1~4 (mAP/P/R/smoke AP) | ✅ TRAINING_LOG 실측값으로 완성 |

→ **텍스트·표 기준으로는 KCI 원고 1편이 이미 성립한다.**

## 3. 보강하면 강해지는 것 (삭제된 가중치 필요)

KCI 심사는 보통 다음 그림을 기대한다. 모두 `runs/` 산출물에서 나오는데 가중치(best.pt)가 삭제돼 현재는 표만 있다.

| 그림 | 필요 산출물 | 현재 |
|------|------------|------|
| 혼동행렬(confusion matrix) | best.pt → val() | ❌ 가중치 삭제됨 |
| PR 곡선 / F1 곡선 | best.pt → val() | ❌ |
| 정성 탐지 예시(detection 이미지) | best.pt 추론 | ❌ |
| FP/FPR 직접 지표(비화재보 핵심) | best.pt + NM held-out | ❌ (원래도 미측정) |

### 보강 옵션 (택1)
- **A. 표만으로 투고**: 일부 학술지는 표 중심 ablation 수용. 즉시 투고 가능. 리뷰에서 그림 요청 가능성 있음.
- **B. 핵심 4셀만 재학습**(권장): C1/C3/C4 + 11s 4회 재학습으로 모든 그림·FP/FPR 재생성.
  - 데이터: Kaggle DFire(자격증명 확인 완료) 또는 AIHub 071751(로그인 필요)
  - DFire 사용 시: ~21K장, 4090에서 셀당 ~2-3h, 전체 6-12h. 단일 공개 벤치마크로 통일돼 방법론 더 깨끗(R1/R2 테스트셋 불일치 문제 해소)
  - 단, 재학습 시 절대 수치는 현 TRAINING_LOG(AIHub 175K)와 달라짐 → 원고 표 갱신 필요
